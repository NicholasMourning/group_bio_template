Inspiring figure

Grace Hopper

Grace Hopper was an influential figure in the computer science industry. She is known for shaping the trajectory of early software development. She played a key role in creating one of the first high-level programming languages, COBOL (Common Business Oriented Language). This language democratized computing by making it more comprehensible and accessible to programmers. She also introduced the idea of machine-independent programming and championed the concept of "write once, run anywhere", an ideology that laid the groundwork for the development of compilers. These tools translate human-readable code into instructions that machines can understand.

Favorite tool

Git-hub Desktop

Git-hub desktop is a goated application that reduces the learning curve of Git.

Reasons why I love it:

  • It has buttons to commit, push, pull etc. (reduces the need to memorize commands)
  • Easy to sync changes with team members, it shows who made what changes and when.
  • Helps you learn Git workflow(staging, commiting, branching) without needing a terminal.
